# yup-exec

```
NAME:
   exec - execute external commands

USAGE:
   exec [OPTIONS] COMMAND [ARG...]

      Execute COMMAND with given arguments.

COMMANDS:
   help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --directory value, -C value                      run command in DIRECTORY
   --env value, -e value [ --env value, -e value ]  set environment variable (NAME=VALUE)
   --shell value                                    shell to use for execution
   --use-shell, -s                                  execute command through shell (default: false)
   --ignore-errors                                  ignore command execution errors (default: false)
   --quiet, -q                                      suppress command output (default: false)
   --interactive, -i                                run in interactive mode (default: false)
   --inherit-env                                    inherit environment variables from parent (default: false)
   --help, -h                                       show help
```
